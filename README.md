# Reservation System Front

Interfaz web construida con **Quasar Framework** (sobre **Vue.js**) para un sistema de gestión de reservas.  
Este proyecto sirve como frontend para una API de reservas (por ejemplo, [Reservation System API](https://github.com/Manuel-IA/reservation-system-api)) y como ejemplo de cómo estructuro una SPA en Quasar para casos de uso de negocio.

---

## Objetivos del proyecto

- Proporcionar una base de frontend en **Quasar/Vue** lista para extender.
- Consumir una API REST externa para gestionar reservas, recursos y usuarios.
- Mostrar cómo organizar páginas, layouts y componentes reutilizables en Quasar.
- Servir como proyecto de referencia para paneles administrativos o aplicaciones internas orientadas a gestión.

> La lógica de negocio y el diseño concreto (qué se reserva, roles de usuario, etc.) pueden adaptarse según el proyecto.  
> El enfoque principal aquí es la **estructura del frontend** y su integración con un backend.

---

## Stack técnico

- **Framework:** [Quasar Framework](https://quasar.dev/) (sobre Vue.js)
- **Librería:** Vue.js
- **Estilos:** SCSS + utilidades de Quasar
- **Lenguaje:** JavaScript
- **Gestor de dependencias:** npm o yarn

---
